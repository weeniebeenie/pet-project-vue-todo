# To-Do List

### About:
```
Simple To-do list based on Vue.js framework.

Main functionality:
- Authentication
- To-do: add/delete/filter tasks
- Light/Dark mode
```

### Tools and technologies:
```
Realtime Database
Firebase Authentication
Vuelidate
```

